# Changelog

## 0.1.0

- Initial release
- turbo.json IntelliSense with completions for all top-level and task configuration keys
- Task Explorer sidebar with dependency details, cache status, and run actions
- Package Explorer showing workspace packages and their scripts
- CodeLens for running tasks directly from turbo.json
- Hover documentation for all turbo.json keys, special values, and glob patterns
- Diagnostics for unknown keys, invalid values, deprecated options, and invalid combinations
- Pipeline Visualization webview showing task dependency DAG
- Snippets for common task configurations
